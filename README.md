# NW-themes
themes for R Studio that are a bit more _aesthetically pleasing_, so we can all code in pink <3

# Theme: womeninSTEMwantpink
Based on basti-purple-dark theme ('https://github.com/bastianolea/rstudio_purple_dark_theme), which is based on the base16 Default Dark theme from {rsthemes}, which in turn is based on base16.
